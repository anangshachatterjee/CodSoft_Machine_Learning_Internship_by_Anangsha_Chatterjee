# CodSoft_Machine_Learning_Internship_by_Anangsha_Chatterjee

## Overview

This repository contains the Machine Learning tasks/projects completed during my **Machine Learning Internship at CodSoft**.

The internship focused on applying Machine Learning algorithms to real-world datasets, performing data preprocessing, model training, evaluation, visualization, and comparing multiple algorithms for classification and prediction tasks.

Each task/project demonstrates the complete Machine Learning workflow from data preprocessing to model evaluation.

## Tools and Technologies used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

- Decision Trees

- Gradient Boosting
- Support Vector Machine (Linear SVM)
- Naive Bayes
